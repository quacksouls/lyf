---
title: Images
---

Use HTML or Markdown to insert an image into your document. This HTML code

```html
<img src="haskell.svg" />
```

would render as:

<img src="haskell.svg">

This Markdown code

```md
!["Haskell"](haskell.svg "Haskell")
```

would render as:

!["Haskell"](haskell.svg "Haskell")

Refer to the subsections for more options.
